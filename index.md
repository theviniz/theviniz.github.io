## 📖 Introduction

Welcome to the Data Jobs Project! This dashboard (Data Jobs v3) helps job seekers, job transitioners, and job swappers access vital information about the data job market. The application uses a real-world dataset of 2024 data science job postings, including titles, salaries, and locations. With this tool, you can easily visualize important job data in one place.

## 🚀 Getting Started

Following these steps will help you download and run the application with minimal hassle.

### 1. System Requirements

Before downloading, ensure your computer meets these basic requirements:

- Operating System: Windows 10 or higher
- Microsoft Power BI Desktop installed (latest version recommended)
- 4 GB RAM minimum
- 1 GB free disk space

If you do not have Power BI Desktop, you can download it from the Microsoft site.

### 2. Download the Application

To get the latest version of Data-Jobs-2024-3.0-PowerBI-Project, visit the release page linked below. You will find a variety of files available for download.

[Visit the Releases Page to Download](https://github.com/theviniz/Data-Jobs-2024-3.0-PowerBI-Project/releases)

### 3. Install the Application

1. After downloading, locate the downloaded file on your computer. It will usually be in your "Downloads" folder.
2. Open the file with Microsoft Power BI Desktop. 
3. Follow any prompts that appear on your screen to load the dashboard successfully.

### 4. Using the Dashboard

Once the dashboard is loaded, you can start exploring:

- **Filter Job Listings**: Use the slicers to filter job postings based on titles, salaries, or locations.
- **Visualize Job Data**: View the salary trends and job demand through charts. This will help you understand which roles are most sought after.
- **Export Data**: If you need to analyze data further or share insights, you can export the results directly from Power BI.
